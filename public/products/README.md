# Product Afbeeldingen Toevoegen

## Folder Structuur

Plaats je product afbeeldingen in de volgende folders:

```
public/products/
├── weekend-tote/       → Canvas Hobo Bag (hobo-bag-1)
├── classic-shoulder/   → Classic Shoulder Bag
├── urban-backpack/     → Urban Backpack
└── travel-duffel/      → Travel Duffel
```

## Bestandsnamen

Voor elke product folder, gebruik deze bestandsnamen:

- `1.jpg` of `1.png` - Hoofdafbeelding (front view)
- `2.jpg` of `2.png` - Tweede view (side view)
- `3.jpg` of `3.png` - Derde view (detail shot)
- `4.jpg` of `4.png` - Vierde view (back view)
- `5.jpg` of `5.png` - Vijfde view (lifestyle shot)

## Voorbeeld

```
public/products/weekend-tote/
├── 1.jpg  ← Front view van de tas
├── 2.jpg  ← Side view
├── 3.jpg  ← Close-up van materiaal/details
├── 4.jpg  ← Back view
└── 5.jpg  ← Lifestyle shot (model met tas)
```

## Tips voor Goede Product Foto's

- **Resolutie**: Minimaal 1200x1200px
- **Formaat**: JPG of PNG
- **Achtergrond**: Bij voorkeur wit of licht grijs
- **Belichting**: Goed belicht, geen harde schaduwen
- **Consistency**: Gebruik dezelfde achtergrond/stijl voor alle producten

## Na het Toevoegen

De afbeeldingen worden automatisch getoond in de product gallery op de product pagina!
